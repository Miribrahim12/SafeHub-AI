# SafeHub AI 🛡️
**Neural Threat Detection for Azerbaijan**

## 💡 Problem
Azərbaycanda sosial mühəndislik və fişinq hücumları artmaqdadır. Mövcud sistemlər Azərbaycan dilinin linqvistik incəliklərini başa düşmür.

## 🚀 Həll
SafeHub AI, Google-un Gemini 1.5 Flash modelini istifadə edərək mesajları Azərbaycan dilində analiz edən ağıllı təhlükəsizlik skaneridir.

## 🛠 Texniki Stack
- **Backend:** Python / Flask
- **Frontend:** TailwindCSS / JetBrains Mono
- **Cloud:** Google Cloud Run (Serverless)

## 🎯 Biznes Potensialı
- B2C: Fərdi istifadəçilər üçün veb alət.
- B2B: Şirkətlər üçün daxili yazışmaların təhlükəsizlik filtri.
- API: Digər tətbiqlərə inteqrasiya imkanı.
